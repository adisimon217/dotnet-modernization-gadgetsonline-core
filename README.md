# GadgetsOnline-DotNet
A sample online shopping app on .NET 10.

# Pre-Requisites 

1. Visual Studio 2019 16.4 or later 
2. .NET 10.0 SDK 

## Quick Start

Simply run the application. The app uses SQLite:

```bash
dotnet run --project GadgetsOnline/GadgetsOnline.csproj
```
